## 安装和使用

### 安装步骤

```bash
# 克隆仓库
git clone https://github.com/kuiningzzzz/example1.git

# 进入项目目录
cd example1

# 安装依赖
npm install

# 运行项目
npm run dev
```

### 配置说明

在项目根目录创建 `.env` 文件：

```env
PORT=3000
DATABASE_URL=your_database_url
```
